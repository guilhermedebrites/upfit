# UpFit Frontend — Project Brief

## O que é
Frontend web e mobile da plataforma UpFit, um sistema fitness gamificado focado em corrida e musculação.

## Objetivo do frontend
Transformar os domínios já implementados no backend em uma experiência utilizável, clara e engajante para:
- registrar treinos
- acompanhar progressão
- participar de grupos
- entrar em desafios
- receber notificações

## Pilares do produto
- Workouts
- Progressão (XP, nível, streak, conquistas)
- Social (grupos, ranking, feed)
- Desafios
- Notificações

## Situação atual
O backend já possui as fases 1 a 7 concluídas:
- Auth
- Workout
- Progression
- Groups
- Challenges
- Notifications

## Próximo objetivo do frontend
Estruturar o projeto frontend e implementar as primeiras telas integradas com o backend real:
1. Auth
2. Home/Dashboard
3. Workouts
4. Progression
5. Groups
6. Challenges
7. Notifications

## Visão de ambientes
- Dev local: URL base única via Nginx `http://localhost:8080`
- Produção: API Gateway / domínio público

## Resultado esperado
Um frontend com memória de projeto consistente, preparado para agentes de IA implementarem features sem inventar contratos ou quebrar alinhamento com o backend real.
